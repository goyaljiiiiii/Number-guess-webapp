# Number Guessing Game

A clean, modern number guessing game built with HTML, CSS, and JavaScript. It provides instant feedback, tracks attempts, and stores your best score locally.

## Features
- Instant feedback with high/low hints
- Attempt counter and best score tracking
- Responsive layout for desktop and mobile
- Share button with clipboard fallback
- Confetti celebration on success

## Project Structure
- `index.html`
- `assets/css/style.css`
- `assets/js/index.js`
- `assets/img/bg.jpg`

## Run Locally
Open `index.html` in any modern browser.

## Customize
- Change the range in `assets/js/index.js`
- Update colors in `assets/css/style.css`
- Replace the font in `index.html`
